对于仅仅是时间离散的DTFT变换，我们可以看到频域仍然是连续的。进行反 DTFT 变换要进行积分运算，非常麻烦，我们喜欢的是类似 DTFT 的求和运算。

后面出现了两个思路:

  1. 将离散的时域进行周期延拓，然后自然可以用 DTFS
  2. 进一步离散频域，然后用离散点求和得到时域点

Any signal that is stored in a computer must be a finite length sequence.

for (n=0; n<N; n++) {
	Xr[n]=Xi[n]=0;
	for (m=0; m<N; m++) {
		w=2*pi*m*n/N;
		if (forward) w=-w;
		wr=cos(w); wi=sin(w);
		Xr[n]=Xr[n]+wr*xr[m]-wi*xi[m];

}

注意: 低频分量和高频分量的表征

DFT 变换中，X[0],X[1],…,X[N−1]X[0], X[1], \ldots, X[N-1]X[0],X[1],…,X[N−1] 表示 NNN 个频率成分 ([0,2π][0, 2\pi][0,2π] 分布)

其中最低频率成分为 X[0]X[0]X[0]， 最高频率成分为 X[N/2]X[N/2]X[N/2] (通过采样定理可以很好理解)，超过 N/2N/2N/2 记为负数频率成分，下图很清晰地表述这一点，但下图没有表示清楚 DFT 的关于 N/2N/2N/2 点对称的特点，这里不要在意。

因此我们把 N/2N/2N/2 对应的频率称为 Nyquist frequency. This is the highest frequency component that should exist in the input series for the DFT to yield “uncorrupted” results. More specifically if there are no frequencies above Nyquist the original signal can be exactly reconstructed from the samples.

如果我们想把最高频率移动到两边，把最低频率移动到中心，怎么做 ?

(注: 这个需求在二维傅里叶变换中会出现)

X′[n]=X[n−N/2]X'[n] = X[n-N/2]X′[n]=X[n−N/2]

  1. 原始信号获得对应的频谱后，再将低频搬移到中心

  2. 根据傅里叶变换的移位性质:

    x[m]ej2πmn0/N⇔X[n−n0]x[m]e^{j2\pi mn_0/N} \Leftrightarrow X[n-n_0]x[m]ej2πmn0​/N⇔X[n−n0​]

    当 n0=N/2n_0 = N/2n0​=N/2 ，我们有:

    x[m]ejπm=x[m](−1)m=F−1[X[n−N/2]]x[m] e^{j\pi m} = x[m] (-1)^m = \mathcal {F}^{-1} [X[n-N/2]]x[m]ejπm=x[m](−1)m=F−1[X[n−N/2]]

    也就是说，如果原始信号奇数序列对应的值改变正负号，偶数序列不变，直接进行傅里叶变换，可以得到低频中心化的频谱
