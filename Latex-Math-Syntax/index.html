1.3 上标和下标

上标 用 ^ 表示, 下标 用 _ 表示

比如： x^2_i -> xi2x^2_ixi2​

注意: 默认情况下，上下标符号只对下一组起作用。一个组即单个字符或者使用 {...}包裹起来的内容。也就是说，使用 10^10，会得到 101010^101010， 而 10^{10} 才是 101010^{10}1010。同时，大括号还能消除二义性，比如 x^5^6 将得到一个错误，必须使用大括号来界定 ^ 的结合性， 比如 {x^5}^6: x56{x^5}^6x56 或者 x^{5^6}: x56x^{5^6}x56

1.4 括号

  1. 小括号与方括号： 使用原始的 (), [] 即可，比如: (2+3)[4+4]: (2+3)[4+4](2+3)[4+4](2+3)[4+4]
  2. 大括号: 由于大括号 {} 被用来分组，因此使用 \lbrace 和 \rbrace 表示大括号。 比如: \lbrace a*b \rbrace: {a∗b}\lbrace a*b \rbrace{a∗b}
  3. 尖括号：使用 \langle 和 \rangle 表示，如 \langle x \rangle : ⟨x⟩\langle x \rangle⟨x⟩
  4. 上取整: 使用 \lceil 和 \rceil 表示，如 \lceil x \rceil: ⌈x⌉\lceil x \rceil⌈x⌉
  5. 下取整: 使用 \lfloor 和 \rfloor 表示，如 \lfloor x \rfloor: ⌊x⌋\lfloor x \rfloor⌊x⌋

1.5 求和和积分

  1. \sum 表示求和符号，下标和上标分别表示求和的下限和上限，比如 \sum_1^n: ∑1n\sum_1^n∑1n​

  2. \int 表示积分符号，上下标表示积分的上下限，比如 \int_1^\infty: ∫1∞\int_1^\infty∫1∞​

    二重积分: \iint: ∬1∞\iint_1^{\infty}∬1∞​

    线积分: \oint: ∮\oint∮

  3. \prod 表示求积符号 ∏\prod∏

  4. 集合符号(好像不是的，奇怪)

    并集: \bigcup: ⋃\bigcup⋃

    交集: \bigcap: ⋂\bigcap⋂

1.6 分式和根式

  1. 分式的表示: \frac {}{}，比如 \frac{a+1}{b+1}: a+1b+1\frac {a+1}{b+1}b+1a+1​

    还是注意: 如果分子分母不是的单个字符，请用 {...} 来分组

    或者 \over 分隔一个组的前后两部分， 比如 {a+1 \over b+1}: a+1b+1{a+1 \over b+1}b+1a+1​

  2. 根式使用 \sqrt[]{} 来表示

    比如 \sqrt[4]{\frac xy}: xy4\sqrt[4]{\frac xy}4yx​​

1.7 字体

  1. 使用 \mathbb 或者 \Bbb 显示黑板粗体字，经常用来表示实数、整数、有理数、复数的大写字母(只有大写有效)

    比如 \mathbb {CHNQRZ}: CHNQRZ\mathbb {CHNQRZ}CHNQRZ

  2. 使用 \mathbf 显示黑体字

    比如 \mathbf {abcABC}: abcABC\mathbf {abcABC}abcABC

  3. 使用 \mathtt 显示打印机字体

    比如: \mathtt {abcABC}: abcABC\mathtt {abcABC}abcABC

  4. 使用 \mathrm 显示罗马字体

    比如： \mathrm {abcABC}: abcABC\mathrm {abcABC}abcABC

  5. 使用 \mathscr 显示手写体(只有大写有效)

    比如: \mathscr {ABCDEF}: ABCDEF\mathscr {ABCDEF}ABCDEF

  6. 使用 \mathfrak 显示 Fraktur 字母 : BM\mathfrak {BM}BM

  7. \mathcal {ABCDEFGM} : ABCDEFGM\mathcal {ABCDEFGM}ABCDEFGM

列样式使用 c 表示居中, l 表示左对齐, r 表示右对齐，使用 | 表示一条竖线

4. 一些书写建议

4.1 不要在指数或者积分中使用 \frac

在指数或者积分表达式中使用\frac会使表达式看起来不清晰，因此在专业的数学排版中很少被使用。应该使用一个水平的/来代替，效果如下：

4.2 使用 \mid 代替 | 作为分隔符

符号|作为分隔符时有排版空间大小的问题，应该使用\mid代替。效果如下：
